# Vue-Plate

Vue.js 2 & Webpack 2 boilerplate. Before you begin please install Yarn. https://yarnpkg.com/lang/en/docs/install/

![alt tag](https://media.giphy.com/media/3o6UBhjHobLFgEmrJu/giphy.gif)]

# Setup
````javascript
$ yarn install
````

# Development
````javascript
$ yarn run dev
````

# Production
````javascript
$ yarn run prod
````
